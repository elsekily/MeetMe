Graduation Project: (IoT based smart agriculture solution for improving irrigation water efficiency usage)

The idea is enhancing irrigation water and achieve better utilization efficiency than conventional irrigation systems. Project employs IoT based solution that captures data from the agricultural land using relevant sensors and embedded systems. The captured data is then sent to a cloud server to decide the quantity and timing of irrigation water. We used Raspberry PI as a local broker and hotspot to gather data from NodeMCUs and ATMega16 then send it to our broker deployed on a virtual machine on Azure then Subscribe it and save into PostgreSQL database and Send to ML then get and publish the water amount to the control devices that will irrigate the land. Project ranked the third place at JAC-ECC, E-just in DEC, 2018. ITIDA Fund the project with 4000EGP. Grade: A (Excellent).

Workers Management:

An Arabic web application to manage and store workers/employees and their Attendance and Leaving times then calculating their working duration in hours and their salaries. Using ASP.Net MVC 5, HTML, CSS, Ajax and jQuery.

Nearest ICU Unit:

The idea is to help patients’ families to find the nearest and empty ICU unit by an Android app that captures the location of the user then send it to ASP.Net Core Restful API via HTTPs, which searches for the nearest empty hospital in the Database, and returns its Location.

Gas Leakage Alarm:

Using Arduino and ESP8266 for measuring Gas in the air if it exceeds the percentage limit ESP will trigger a HTTPs request to IFTTT, which handles the request and send alarm SMS to the user’s mobile phone. Project ranked the First place at Pharos Community projects Competition 2018.

Distance Measuring (Assembly language):

Using ATMega16, Ultrasonic and 16X2 LCD. Measuring the Time takes ultrasonic wave to hit the object using Interruptus and timer then convert it to ASCII String to display the distance on the LCD.
